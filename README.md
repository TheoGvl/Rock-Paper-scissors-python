# Rock, Paper, Scissors 

Μια μοντέρνα, ψηφιακή εκδοχή του κλασικού παιχνιδιού «Πέτρα, Ψαλίδι, Χαρτί» με Γραφικό Περιβάλλον (GUI). Ο χρήστης παίζει ενάντια στον υπολογιστή, ενώ η εφαρμογή κρατάει το σκορ σε πραγματικό χρόνο! Το project αναπτύχθηκε για την εξάσκηση στη λογική των παιχνιδιών  και στη διαχείριση συμβάντων  μέσω της Python και της Flet.

## Τι ακριβώς κάνει;
* **Άμεση Διάδραση:** Διαθέτει τρία κουμπιά επιλογής (🪨, ✂️, 📜) πρώτη φορα έκανα χρήση emoji. Με το πάτημα ενός κουμπιού, ο υπολογιστής κάνει αμέσως τη δική του τυχαία επιλογή.
* **Αυτόματη Κρίση:** Το πρόγραμμα συγκρίνει τις δύο επιλογές, βρίσκει τον νικητή βάσει των κλασικών κανόνων και εμφανίζει το αποτέλεσμα με το αντίστοιχο χρώμα Πράσινο για Νίκη, Κόκκινο για Ήττα, Κίτρινο για Ισοπαλία.
* **Καταγραφή Σκορ:** Κρατάει δυναμικά το σκορ των νικών ανάμεσα στον παίκτη και τον υπολογιστή για όση ώρα είναι ανοιχτή η εφαρμογή.


**Βασικές προγραμματιστικές τεχνικές:**
1. **Λογική Παιχνιδιού (If/Elif/Else):** Χρήση πολλαπλών συνθηκών με λογικούς τελεστές (`or`, `and`) για να καλυφθούν όλα τα πιθανά σενάρια νίκης, ήττας και ισοπαλίας.
2. **Διαχείριση Δεδομένων (Data Attributes):** Χρήση της ιδιότητας `data` στα κουμπιά, ώστε μία μόνο συνάρτηση (`paixe`) να αναγνωρίζει δυναμικά ποιο από τα τρία κουμπιά πατήθηκε (`e.control.data`).
3. **Διαχείριση Μνήμης (Nonlocal State):** Διατήρηση και ανανέωση του σκορ έξω από την κύρια συνάρτηση του παιχνιδιού, για να μην μηδενίζεται σε κάθε γύρο.
4. **Τυχαιότητα:** Χρήση της `random.choice()` για να συμπεριφέρεται ο υπολογιστής σαν αληθινός, απρόβλεπτος αντίπαλος.
